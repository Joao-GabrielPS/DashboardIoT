# 💧 Hydro Monitor

Uma dashboard desenvolvida para monitorar o nível de uma caixa d'água em tempo real utilizando um **ESP** (ESP8266/ESP32) e um **sensor ultrassônico**. O sistema realiza a leitura da distância entre o sensor e a água, calcula o nível disponível e exibe as informações de forma simples e intuitiva.

## 🚀 Funcionalidades

- 📡 Leitura em tempo real do nível da água.
- 📊 Exibição do percentual de preenchimento da caixa.
- ⚡ Atualização automática dos dados.
- 📱 Interface responsiva para acesso pelo computador ou celular.

## 🛠️ Tecnologias Utilizadas

- ESP8266
- Sensor Ultrassônico HC-SR04
- HTML, CSS e JavaScript
- Comunicação via Wi-Fi
- Dashboard Web

## ⚙️ Como Funciona

1. O sensor ultrassônico mede a distância até a superfície da água.
2. O ESP processa a leitura e calcula o nível da caixa.
3. Os dados são enviados para a dashboard.
4. A interface exibe o nível em tempo real para acompanhamento.
